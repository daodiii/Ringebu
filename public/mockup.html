Approach A — Recommended

CSS Columns Masonry

Cards flow top→bottom in 3 columns. Each card is exactly as tall as its content. Zero wasted space. The varied content (some have prices, some don't) creates natural height differences — a mosaic effect without any CSS Grid hacks.

Approach B

Flex Rows with Varied Splits

Independent flex rows with different width ratios. No grid, no row-spans, no gaps. Each row has its own column split (55/45, 33/33/33, 40/60, 35/30/35) so every card gets a unique width. Height is always content-driven.

Approach C

Full-Width Editorial Sections

Each treatment is a full-width section with alternating internal layouts. More like a magazine article. Very long page, but maximum readability and zero wasted space. Content flows left-to-right in alternating splits.
